# QuantCore vs Backtrader vs Vn.py vs AKQuant 功能对比

**日期**: 2026-04-06  
**版本**: QuantCore v0.2.0

---

## 1. 框架概述

### QuantCore
- **定位**: A 股原生全栈式量化交易框架
- **优势**: Rust 核心引擎性能卓越，Python 接口易用，A 股原生支持
- **状态**: 活跃开发中 (v0.2.0)
- **代码量**: 6,000+ 行
- **特色**: 多策略组合、专业可视化、T+1 支持
- **理念**: 以探索性研究为主，未来支持实盘交易

### Backtrader
- **定位**: Python 纯量化回测框架
- **优势**: 功能成熟、生态完善、文档丰富
- **状态**: 成熟稳定 (v1.9.78.123)
- **代码量**: 50,000+ 行
- **特色**: 策略组合、数据分析、社区活跃

### Vn.py (VeighNa)
- **定位**: Python 量化交易框架（侧重实盘）
- **优势**: 实盘接口丰富、社区活跃、功能全面
- **状态**: 活跃开发中 (v3.9.0+)
- **代码量**: 100,000+ 行
- **特色**: 实盘交易、多交易所支持、风控系统

### AKQuant (AKShare)
- **定位**: Python 金融数据接口库
- **优势**: 数据源丰富、开源免费
- **状态**: 活跃开发中
- **代码量**: 100,000+ 行（含数据接口）
- **特色**: 数据采集、A 股数据全面

### 5.2 代码示例对比

**相同策略：双均线交叉**

#### QuantCore
```python
from quantcore import *

class DualMA(Strategy):
    def on_bar(self, bar, engine):
        self.prices.append(bar.close)
        if len(self.prices) < 20:
            return
        fast_ma = sum(self.prices[-5:]) / 5
        slow_ma = sum(self.prices[-20:]) / 20
        if fast_ma > slow_ma:
            self.buy(bar.symbol, bar.close, 1000)
```

#### Backtrader
```python
import backtrader as bt

class DualMA(bt.Strategy):
    def __init__(self):
        self.ma_fast = bt.ind.EMA(period=5)
        self.ma_slow = bt.ind.EMA(period=20)
    
    def next(self):
        if self.ma_fast > self.ma_slow:
            self.buy(size=1000)
```

