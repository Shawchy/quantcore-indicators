## 🛠️ 技术细节

### 行业分配逻辑
```python
# 使用股票代码后两位作为索引，使同一行业的股票相对集中
code_num = int(stock.code[-2:]) if stock.code[-2:].isdigit() else random.randint(0, len(INDUSTRIES)-1)
stock.industry = INDUSTRIES[code_num % len(INDUSTRIES)]
```
