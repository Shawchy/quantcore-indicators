## 🎯 核心定位

**QuantCore - A 股原生全栈式量化交易框架**

### 一句话介绍

QuantCore 是一个面向 A 股市场的全栈式量化交易框架，以**探索性研究**为主，提供**高性能回测**、**多策略组合**和**专业可视化**功能，未来将支持**实盘交易**。

---

## 🔑 三大核心特征

### 1. A 股原生 🇨🇳

**深度优化 A 股交易规则**，不是简单的适配，而是从底层设计就为 A 股量身打造。

**具体体现**：
- ✅ **T+1 交易规则**：当日买入不可当日卖出，完全符合 A 股制度
- ✅ **涨跌停限制**：支持 10%/20% 的涨跌停板撮合逻辑
- ✅ **交易费用结构**：佣金、印花税、过户费完整支持
- ✅ **交易时间**：A 股交易时段（9:30-11:30, 13:00-15:00）
- ✅ **最小交易单位**：100 股起买，100 股整数倍

**为什么重要**：
- 很多国际框架（如 Backtrader）对 A 股支持不完善
- 国内框架（如 Vn.py）主要面向期货，股票是"适配"而非"原生"
- QuantCore 从设计之初就为 A 股考虑，规则更准确、使用更便捷

### 2. 探索研究 🔬

**为量化研究而设计**，帮助用户快速验证策略想法。

**具体体现**：
- ⚡ **高性能回测**：Rust 引擎，比 Backtrader 快 10-20 倍
  - 10 年日线数据回测：5 秒 vs 50 秒
  - 多策略并行：30 秒 vs 8 分钟
  - 内存占用降低 80%
  
- 🎯 **快速验证**：
  - 简洁的 API 设计，3-5 行代码运行回测
  - 丰富的技术指标库（MA、MACD、RSI、BOLL 等 10+ 指标）
  - 参数优化工具（网格搜索、随机搜索）
  
- 📊 **专业分析**：
  - 完整的绩效评估体系（夏普比率、最大回撤、年化收益等）
  - 5+ 种专业图表一键生成
  - 策略对比分析

**为什么重要**：
- 量化研究需要快速迭代，性能是关键
- 研究者关注策略逻辑，而非框架细节
- 可视化帮助理解策略表现，辅助决策

### 3. 全栈式 🧩

**从研究到回测的一站式解决方案**，未来将扩展至实盘交易。

**当前能力**（v0.2.0）：
```
研究阶段
├── 数据加载（Baostock、CSV）
├── 策略开发（Python 友好接口）
├── 参数优化（网格搜索、随机搜索）
└── 策略验证（高性能回测）

分析阶段
├── 绩效评估（完整指标体系）
├── 可视化（资金曲线、回撤、收益分布）
├── 策略对比（多策略对比图）
└── 风险分析（最大回撤、波动率）

组合管理
├── 多策略组合
├── 权重配置
├── 资金分配
└── 组合绩效
```

**未来规划**（v0.4.x+）：
```
实盘交易
├── 交易接口（股票、期货）
├── 风控系统（事前、事中、事后）
├── 订单管理（限价、市价、条件单）
└── 持仓管理（实时监控、自动调仓）
```

**为什么重要**：
- 用户不需要在多个工具间切换
- 从研究到实盘的平滑过渡
- 降低量化交易门槛

## 🎓 适用场景

### ✅ 推荐使用

#### 1. 量化研究
- **场景**：验证新的交易策略想法
- **优势**：高性能回测，快速迭代
- **示例**：
  ```python
  # 3 行代码运行回测
  portfolio = StrategyPortfolio(initial_capital=1000000)
  portfolio.add_strategy("MACD", MACDStrategy(), weight=1.0)
  result = portfolio.run(bars, tplus1=True)
  ```

#### 2. 策略优化
- **场景**：调整策略参数，寻找最优配置
- **优势**：参数优化工具，网格搜索/随机搜索
- **示例**：
  ```python
  # 网格搜索最优参数
  optimizer = GridSearch()
  results = optimizer.optimize(
      strategy_class=MACDStrategy,
      param_grid={'fast_period': [5, 10, 15], 'slow_period': [20, 30, 40]}
  )
  ```

#### 3. 多策略组合
- **场景**：分散投资，降低风险
- **优势**：权重配置，组合绩效分析
- **示例**：
  ```python
  portfolio = StrategyPortfolio(initial_capital=1000000)
  portfolio.add_strategy("MACD", MACDStrategy(), weight=0.4)
  portfolio.add_strategy("RSI", RSIStrategy(), weight=0.3)
  portfolio.add_strategy("BOLL", BOLLStrategy(), weight=0.3)
  result = portfolio.run(bars)
  ```

#### 4. 量化教学
- **场景**：学习量化交易，实践策略开发
- **优势**：简洁 API，完整文档，丰富示例
- **示例**：
  ```python
  # 10 行代码实现双均线策略
  class DualMAStrategy(Strategy):
      def on_bar(self, bar, engine):
          if fast_ma > slow_ma:
              engine.buy(bar.symbol, bar.close, 1000)
          elif fast_ma < slow_ma:
              engine.sell(bar.symbol, bar.close, 1000)
  ```

### ❌ 不推荐（当前版本）

#### 1. 实盘交易
- **原因**：实盘接口尚未实现（计划 v0.4.x）
- **替代方案**：使用 Vn.py 进行实盘交易

#### 2. 期货交易
- **原因**：目前仅支持 A 股，期货支持待开发
- **替代方案**：使用 Vn.py 进行期货交易

#### 3. 高频交易
- **原因**：框架设计面向中低频策略
- **替代方案**：使用专业高频交易系统
