## 📝 前端代码文档化建议

### 1. 使用 TypeScript 类型系统

```typescript
/**
 * 基金基本信息接口
 * 
 * @example
 * ```typescript
 * const fund: FundBasicInfo = {
 *   fundCode: '000001',
 *   fundName: '华夏成长混合',
 *   netValue: 1.234,
 *   growthRate: 0.05
 * }
 * ```
 */
interface FundBasicInfo {
  /** 基金代码（6 位数字） */
  fundCode: string
  
  /** 基金名称 */
  fundName: string
  
  /** 净值（元） */
  netValue: number
  
  /** 增长率（小数，0.05 表示 5%） */
  growthRate: number
}
```

### 2. 使用 JSDoc 注释

```typescript
/**

 * @returns 历史净值数据
 * 
 * @throws {ApiError} 当 API 请求失败时
 * 
 * @example
 * ```typescript
 * const history = await getFundHistory('000001', '2024-01-01', '2024-12-31')
 * ```
 */
async function getFundHistory(
  fundCode: string,
  startDate: string,
  endDate: string
): Promise<FundHistoryData> {
  // 实现代码
}
```

### 3. 使用 Git Commit Message

```bash
# ✅ 好的 Commit Message

- 实现基金列表组件
- 添加收益率排序
- 支持基金详情跳转

- 增加超时时间到 60 秒

更新代码
修复 bug
添加功能
