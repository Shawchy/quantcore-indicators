## 已知问题

### 1. Polars 1.x 破坏性变更

**问题**: Polars 1.0 有一些破坏性变更

**解决方案**:
```python
# 旧代码
df.filter(pl.col("a") > 1)

# 新代码（推荐）
df.filter(pl.col("a") > 1)  # 仍然可用
# 或使用新语法
df.filter(pl.col("a").gt(1))
```

**影响**: 低 - 大部分代码兼容

### 2. Pydantic 2.10 严格模式

**问题**: Pydantic 2.10 默认启用严格模式

**解决方案**:
```python
# 如果需要宽松模式
class Settings(BaseSettings):
    model_config = ConfigDict(strict=False)
```

**影响**: 中 - 可能需要调整部分模型定义

### 3. Numba 0.60 LLVM 要求

**问题**: Numba 0.60 需要 LLVM 14+

**解决方案**: Windows 用户通常无需手动安装
- 如遇问题，降级到 numba 0.59: `pip install numba==0.59.0`

**影响**: 低 - 预编译包已包含 LLVM
