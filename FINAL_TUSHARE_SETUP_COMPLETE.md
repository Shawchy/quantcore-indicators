2. **设置 Token**:
   ```

   请帮我保密这个 Token
   ```

3. **开始使用**:
   ```
   帮我查询平安银行的股票信息
   获取上证指数的日线数据
   显示银行板块的所有股票
   ```

### 在 Python 中使用

```python
import tushare as ts

# 初始化
ts.set_token('15e846af88b6c0d82517edb3118bdbb55374780630d7e53c8c5b4105')
pro = ts.pro_api()

# 获取股票列表
df = pro.stock_basic(list_status='L')
print(f"共{len(df)}只股票")

# 获取日线行情
df = pro.daily(ts_code='000001.SZ', start_date='20240101')
print(df.head())

# 获取指数数据
df = pro.index_daily(ts_code='000001.SH')
print(df.head())
```
