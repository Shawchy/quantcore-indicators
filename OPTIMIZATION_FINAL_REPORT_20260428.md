### 1.1 financial_llm_router.py - 信号融合实现

**实现内容**:
- ✅ 双模型信号融合逻辑（数值 60% + 情感 40%）
- ✅ 信号提取算法（关键词匹配）
- ✅ 综合置信度计算
- ✅ 操作建议生成
- ✅ 集成到 QwenAssistant 服务

**融合算法**:
```python
信号评分 = {
    "bullish": 数值看涨 * 0.6 + 情感看涨 * 0.4,
    "bearish": 数值看跌 * 0.6 + 情感看跌 * 0.4,
    "neutral": 数值中性 * 0.6 + 情感中性 * 0.4
}
最终信号 = max(信号评分)
综合置信度 = 最大信号得分 / 总得分
```

**操作建议**:
- 看涨 + 置信度 ≥ 60% → 轻仓试多
- 看跌 + 置信度 ≥ 60% → 减仓或观望
- 其他 → 信号不明确，建议观望
