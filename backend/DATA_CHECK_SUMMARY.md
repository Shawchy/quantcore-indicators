   - 解决：调用 `get_sector_list()` 拉取数据

2. **technical_indicators 表为空**
   - 影响：技术指标分析不可用
   - 解决：运行指标计算任务

3. **chip_data 表为空**
   - 影响：筹码分析不可用
   - 解决：调用筹码数据接口

### P2 - 优化建议

1. **数据归档自动化**
   - 建议：定时任务自动归档 90 天前数据

   - 收益：及时发现数据异常
