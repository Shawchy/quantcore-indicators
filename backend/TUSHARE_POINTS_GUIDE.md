## ⚙️ 配置方法

### 方法一：修改 .env 文件（推荐）

打开 `backend/.env` 文件：

```env
# 设置您的积分（默认 120 分）
TUSHARE_POINTS=120
```

根据您的实际积分修改：

```env
# 如果您有 5000 分
TUSHARE_POINTS=5000

# 如果您有 10000 分
TUSHARE_POINTS=10000
```

### 方法二：使用环境变量

**Windows PowerShell**:
```powershell
$env:TUSHARE_POINTS="5000"
```

**Linux/Mac**:
```bash
export TUSHARE_POINTS=5000
```

### 方法三：修改配置文件

打开 `backend/app/config.py`:

```python
TUSHARE_POINTS: int = 5000  # 修改为您的实际积分
```

## ⚠️ 注意事项

### 1. 积分只作为门槛

- ✅ 达到积分即可调用，不消耗积分
- ✅ 积分不扣费，只作为权限控制
- ⚠️ 积分有效期 1 年

### 2. 积分有效期

- 积分有效期 1 年
- 可通过签到、社区贡献续期
- 过期后需要重新获取

### 3. 调用频率限制

积分越高，调用频率上限越高：

| 积分范围 | 每分钟调用 | 每日调用 |
|---------|----------|---------|
| 120 分 | 20 次 | 500 次 |
| 5000 分 | 100 次 | 5000 次 |
| 10000 分 | 500 次 | 50000 次 |

### 4. 自动降级

如果积分不足：
1. Tushare 返回空数据
2. 系统自动切换到 AkShare
3. 用户无感知，业务代码无需修改
