## 🐛 **常见问题排查**

### 问题 1: 后端接口返回错误

**症状**: 500 错误
**排查**:
1. 检查 Tushare Token 是否配置
2. 检查网络连接
3. 查看后端日志

**解决**:
```bash
# 检查 .env 文件
cat backend/.env | grep TUSHARE_TOKEN

# 重启后端服务
python backend/main.py
```

### 问题 2: 前端无法获取数据

**症状**: 页面显示"获取数据失败"
**排查**:
1. 检查后端服务是否启动
2. 检查 API 地址配置
3. 查看浏览器控制台错误

**解决**:
```bash
# 检查前端配置
cat frontend/.env

# 确保 VITE_API_BASE_URL 配置正确
VITE_API_BASE_URL=http://localhost:8000/api/v1
```

### 问题 3: 数据采集时间过长

**症状**: 超过 5 分钟未完成
**排查**:
1. 检查网络状况
2. 检查数据源服务器状态
3. 查看后端日志

**解决**:
- 切换到东方财富数据源
- 减少显示数量（前 20）
