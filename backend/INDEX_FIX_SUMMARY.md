if df is not None and len(df) > 0:

**修改后：**
```python
if df is not None and len(df) >= 2:
    row = df.iloc[-1]  # 最新交易日
    prev_row = df.iloc[-2]  # 前一交易日
    close = float(row.get('close', 0))
    prev_close = float(prev_row.get('close', 0))
    change = close - prev_close  # ✅ 计算涨跌额
    change_pct = (change / prev_close) * 100 if prev_close != 0 else 0.0  # ✅ 计算涨跌幅
```

## 技术细节

### 数据来源
- 数据源：AkShare (ak.stock_zh_index_daily)
- 计算方法：最新收盘价 - 前一交易日收盘价
- 更新频率：实时（每个交易日）

### 计算公式
```python
涨跌额 = close - prev_close
涨跌幅 = (涨跌额 / prev_close) * 100
```

### 边界情况处理
- 数据不足 2 条：不返回数据（需要至少 2 个交易日数据）
- prev_close 为 0：涨跌幅返回 0.0（避免除零错误）
