## 🎯 **实现内容**

### 一、后端 API（已有）

#### 日线行情接口

**接口路径**: `GET /api/v1/stock/kline/{code}`

**参数**:
- `code`: 股票代码（6 位数字）
- `start_date`: 开始日期（YYYY-MM-DD）
- `end_date`: 结束日期（YYYY-MM-DD）
- `adjust`: 复权类型（qfq/hfq/none）
- `priority_load`: 是否优先加载

**返回数据**:
```json
{
  "success": true,
  "data": {
    "status": "complete",
    "data": [
      {
        "date": "2024-03-12",
        "open": 10.87,
        "high": 10.96,
        "low": 10.85,
        "close": 10.94,
        "volume": 754905.58,
        "amount": 824171954.0,
        "turnover_rate": 1.23
      }
    ]
  }
}
```

**数据源**:
- Tushare（120 积分）
- AkShare（免费）
- Baostock（免费备用）

**均线计算**:
```typescript
const calculateMA = (data: KLineData[], period: number) => {
  const result = []
  for (let i = 0; i < data.length; i++) {
    if (i < period - 1) {
      result.push('-')
      continue
    }
    const sum = data.slice(i - period + 1, i + 1)
      .reduce((s, d) => s + d.close, 0)
    result.push(+(sum / period).toFixed(2))
  }
  return result
}
```

### 2. **数据格式化**

```typescript
// 价格：保留 2 位小数
formatPrice: (price: number) => price.toFixed(2)

// 成交量：万手/亿手
formatVolume: (volume: number) => {
  if (volume >= 100000000) return `${(volume / 100000000).toFixed(2)}亿手`
  if (volume >= 10000) return `${(volume / 10000).toFixed(1)}万手`
  return volume.toFixed(0)
}

// 成交额：万/亿
formatAmount: (amount: number) => {
  if (amount >= 100000000) return `${(amount / 100000000).toFixed(2)}亿`
  if (amount >= 10000) return `${(amount / 10000).toFixed(0)}万`
  return amount.toFixed(0)
}
```

```typescript
const handleExport = () => {

    ),
  ].join('\n')
  
  const blob = new Blob(['\ufeff' + csv], { type: 'text/csv;charset=utf-8' })
  const link = document.createElement('a')
  link.href = URL.createObjectURL(blob)
  link.download = `${code}_daily_${new Date().toISOString().split('T')[0]}.csv`
  link.click()
}
```
