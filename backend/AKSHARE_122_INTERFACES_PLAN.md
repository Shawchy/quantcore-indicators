## 📚 16 集系列教程概览

根据实测系列教程，AKShare 122 个股票数据接口涵盖以下主题：

1. **A 股概况 + 股票名录** - 基础数据
2. **实时行情数据** - 沪深京 A 股实时报价
3. **K 线数据** - 日/周/月 K 线
4. **分时数据** - 实时分时行情
5. **资金流向** - 个股/板块资金流
6. **财务指标** - ROE、毛利率、营收增长率
7. **估值指标** - 市盈率、市净率、市销率
8. **市场情绪** - 涨停跌停、创新高/新低
9. **板块行情** - 行业/概念板块
10. **龙虎榜** - 营业部席位数据
11. **股东结构** - 十大股东/流通股东
12. **融资融券** - 两融数据
13. **宏观指标** - 巴菲特指标、市场拥挤度
14. **特色数据** - 北向资金、南向资金

### P0+ - 市场情绪监控（新增）

#### 1. 涨停股池接口 ⭐⭐⭐⭐⭐

**API**: `stock_zt_pool_em()`

**功能**: 获取指定日期的涨停股票池

**数据字段**:
- 代码、名称、涨跌幅、最新价
- 成交额、流通市值、总市值
- 换手率、封板资金
- 首次封板时间、最后封板时间
- 炸板次数、涨停统计、连板数
- 所属行业

**使用场景**:
- 涨停板复盘
- 连板股追踪

#### 4. 创新高/新低统计接口 ⭐⭐⭐⭐⭐

**API**: `stock_a_high_low_statistics()`

**功能**: 获取不同市场的创新高/新低股票数量统计

**数据字段**:
- date: 交易日
- close: 相关指数收盘价
- high20/low20: 20 日新高/新低数量
- high60/low60: 60 日新高/新低数量

**参数**:
- symbol: "all"（全部 A 股）、"sz50"（上证 50）、"hs300"（沪深 300）、"zz500"（中证 500）

**使用场景**:
- 市场趋势判断

## 🔧 技术实现建议

### 1. 数据模型定义

```python
@dataclass
class LimitUpStock:
    """涨停股"""
    code: str
    name: str
    change_pct: float
    latest_price: float
    turnover_rate: float
    limit_up_count: int  # 连板数
    first_limit_time: str
    last_limit_time: str
    industry: str

@dataclass
class LimitDownStock:
    """跌停股"""
    code: str
    name: str
    change_pct: float
    latest_price: float
    continuous_limit_down: int  # 连续跌停
    open_count: int  # 开板次数

@dataclass
class HighLowStatistics:
    """创新高/新低统计"""
    date: str
    index_close: float
    high_20: int
    low_20: int
    high_60: int
    low_60: int
    high_120: int
    low_120: int

@dataclass
class FinancialIndicator:
    """财务指标"""
    code: str
    report_date: str
    roe_weighted: float
    gross_margin: float
    net_margin: float
    revenue_growth: float
    net_profit_growth: float
    asset_liability_ratio: float

@dataclass
class TopShareholder:
    """十大股东"""
    code: str
    shareholder_name: str
    shareholder_type: str
    hold_amount: float
    hold_ratio: float
    change_type: str  # 增持/减持/新进/退出
```

### 2. 缓存策略

| 数据类型 | 缓存时间 | 说明 |
|----------|----------|------|
| 涨停股池 | 1 天 | 历史数据不变 |
| 跌停股池 | 1 天 | 历史数据不变 |
| 创新高/新低 | 1 天 | 盘后更新 |
| 财务指标 | 1 周 | 季报更新 |
| 股东数据 | 1 周 | 季报更新 |
| 北向资金 | 1 天 | 每日更新 |
| 龙虎榜 | 1 天 | 每日更新 |

## ⚠️ 注意事项

### 1. 数据时效性
- 涨停/跌停数据仅在交易日有更新
- 非交易时段返回最近交易日数据
- 财务数据按季度更新

### 2. 性能考虑
- 涨停股池数据量小（通常<100 条）
- 创新高/新低统计为汇总数据
- 建议盘后批量缓存

### 3. 数据源限制
- 东方财富数据源为主
- 部分历史数据可能不完整
- 建议多数据源验证
